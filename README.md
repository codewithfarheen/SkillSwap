# SkillSwap – Skill Exchange Platform

A full-stack web application where users can share their skills and request skills from others.

---

## 🚀 Features

* User Signup & Login Authentication
* Post Skills with Title & Description
* View All Available Skills
* Request Skills from Other Users
* Prevent Duplicate Requests
* Clean and Responsive UI

---

## 🛠 Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Python (Flask)

**Database:**

* SQLite

## ⚙️ How to Run Locally

1. Clone the repository
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
python app.py
```

4. Open in browser:

```
http://127.0.0.1:5000
```
